# Workforce Management Data Warehouse
A data warehouse system for monitoring employee performance, attendance, and productivity analysis using star schema design

## Project Description
This system integrates employee data, attendance, tasks, and salary information to generate insights about:

Employee performance per cluster
Task completion rates
Attendance patterns
Salary increase and promotion predictions

Note: All data is dummy/synthetic for demonstration purposes
